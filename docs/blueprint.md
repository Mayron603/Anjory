# **App Name**: Anjory Store

## Core Features:

- User Authentication: Allow users to create accounts, log in, and manage their profiles.
- Product Catalog: Display available products (Adesivos criativos, Album de Figurinhas, Bloco de notas, Caderno Personalizado, UNO Personalizado Vala Bulbos Medio, Vela Bulbos Mini, Vela Pote de Vidro) with images, descriptions, and prices.
- Shopping Cart: Enable users to add products to a shopping cart and adjust quantities.
- Checkout Process: Guide users through a secure checkout process, including address input, shipping options, and payment.
- Order Management: Allow users to view their order history and track shipments.

## Style Guidelines:

- Primary color: Light blue (#B0E2FF) to create a calm and inviting atmosphere.
- Background color: Very light blue (#E6F7FF) to provide a clean and airy feel.
- Accent color: Slightly darker blue (#87CEEB) to highlight key elements and CTAs.
- Body and headline font: 'PT Sans' for a modern and readable experience.
- Use elegant, simple icons to represent different product categories and functionalities.
- Clean, minimalist layout with clear product presentation, inspired by the provided image.
- Subtle animations for product transitions and interactions to enhance user engagement.
